# 🍊 감귤마켓 SNS 프로젝트

## 📖 프로젝트 소

감귤마켓은 감귤 판매자들이 자신의 상품을 등록하여 홍보할 수 있는 SNS 플랫폼입니다. 오직 감귤만 상품으로 업로드할 수 있으며, 상품 등록 외에도 일상을 공유하고 다른 사용자들과 소통할 수 있는 다양한 기능을 제공합니다.

### ✨ 주요 기능

- 🍊 감귤 상품 등록 및 홍보
- 📝 게시글 작성 및 공유 (텍스트, 이미지)
- 👥 사용자 팔로우/언팔로우
- ❤️ 게시글 좋아요
- 💬 댓글 작성
- 🔍 사용자 검색
- 💌 메시지 기능
- 👤 프로필 관리

## 🛠️ 기술 스택

### Frontend

- HTML5
- CSS3
- JavaScript (ES6+)

### Backend

- API 제공 (백엔드 구현 불필요)
